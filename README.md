# 📊 SuperStore Sales Dashboard

## 📌 Overview
The **SuperStore Sales Dashboard** is an interactive data visualization project designed to analyze sales performance, profitability, and customer behavior across different regions, categories, and time periods.

This dashboard provides key insights that help in making data-driven business decisions.

## 🚀 Features

- 📈 **Sales Overview**
  - Total Sales: **816K**
  - Total Profit: **91K**
  - Total Orders: **3K**
  - Quantity Sold: **12K**

- 🌍 **Regional Analysis**
  - Profit comparison across regions (West, East, South, Central)

- 🛍️ **Product Insights**
  - Sales distribution by different products

- 🧩 **Category Analysis**
  - Sales breakdown by categories (Furniture, Office Supplies, Technology)

- 📅 **Time-Based Trends**
  - Monthly sales performance visualization

- 🎯 **Interactive Filters**
  - Payment Mode (Cards, COD, Online)
  - Ship Mode (First Class, Same Day, Second Class, Standard Class)
  - Customer Segments (Consumer, Corporate, Home Office)

- Order details
- Sales and profit data
- Product categories
- Customer segments
- Shipping modes
- Regional information

## 📊 Key Insights

- 📌 The **West region** generates the highest profit.
- 📌 Sales show a **steady increase towards the end of the year**.
- 📌 Certain products contribute significantly more to overall revenue.
- 📌 Customer segments show varied purchasing behavior patterns.
